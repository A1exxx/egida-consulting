# Эгида — лендинг + маркетинг-пакет

Лендинг консалтинга (бухгалтерия/налоги + кадрово-юридический консалтинг для работодателя) под трафик из Яндекс.Директа. Аудитория — собственники малого бизнеса / ИП-работодатели, РФ.

## Стек
Vanilla HTML/CSS/JS, без сборки и зависимостей. Шрифты — Google Fonts (Lexend / Source Sans 3 / Source Serif 4). Дизайн: Trust & Authority (navy + тёплая бумага + brass gold).

## Файлы
- `index.html` / `styles.css` / `script.js` — лендинг с квизом.
- `CONTENT.md` — бренд, позиционирование, цены, FAQ (источник правды).
- `MARKETING.md` — стратегия, банк офферов, каналы, разбивка бюджета 20к.
- `DIRECT.md` — Яндекс.Директ: семантика, объявления, гео, цели Метрики, чек-лист запуска.
- `VIDEO-SCENARIOS.md` — 8 сценариев роликов «эксперт на камеру» + памятка съёмки.
- `creatives/` — баннеры для РСЯ.

## Как открыть локально
Просто открыть `index.html` в браузере, либо поднять статику:
```
python -m http.server 8099 --directory .
```

## Приём заявок (важно)
Форма и квиз собирают заявку и открывают **WhatsApp** (`+7 919 187-73-50`) с предзаполненным текстом + дублируют в `localStorage` (ключ `egida_leads`). Это осознанный MVP-вариант «собрать заявки в переписку».

WhatsApp в РФ на 2026 нестабилен — это **временный канал**. Рекомендуемый следующий шаг: подключить Telegram-бота/уведомления менеджеру и виджет обратного звонка (см. `MARKETING.md`). Номер для лида и канал меняются в одном месте — константа `WA_PHONE` в `script.js`.

## Открытые вопросы (решить с пользователем)
- Финальное имя бренда (рабочее «Эгида») и домен.
- Финальный канал доставки лидов (Telegram-бот / обратный звонок / форм-сервис).

- Поставить счётчик Яндекс.Метрики + цели перед запуском Директа (см. `DIRECT.md`).
